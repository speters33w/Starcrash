The Little Book of Adventure Game Programming in Java

This source code archive contains the code of The Little Book Of Adventure Game Programming in Java.

The code archive is being released in November 2020.

The book that accompanies is scheduled for release in December 2020.

To receive the latest news and updates, be sure to sign onto our mailing list at: http://www.bitwisebooks.com

Note that this archive includes the code of a framework that you may use to develop your own text adventure games.
The framework is called BIFF (the Bitwise Interactive Fiction Framework).
In this initial release the code of BIFF has not be separated out from the sample games.
You may regard the classes provided with the Starcrash project as representing the BIFF framework.
In later releases, BIFF will be supplied separately from the sample programs and it will be extended and refined with new and improved features.

You may use and modify BIFF for free or commercial purposes.
However, if you use the source code if BIFF please retain the copyright notices in the code.
